# Halfling

## Setting Up a Halfling

First, select the "Halfling" sheet type:

1. Click the **Toggle Controls** button (three vertical dots) in the title bar
2. Click **Configure Sheet**
3. In the **This Sheet** dropdown, select **Halfling**
4. Click **Save Sheet Configuration**

## Halfling Tab

The Halfling tab shows the **Sneak and Hide** skill that Halflings can roll.

## Two-Weapon Fighting

Halflings are natural two-weapon fighters. To set up two-weapon fighting, create two weapon items and configure them with the **2-Weap. Prim.** and **2-Weap. Off** checkboxes.

See [Two-Weapon Fighting](Two-Weapon-Fighting.md) for detailed instructions.
